# Secure Cloud Computing: Deploying a ML Web App on a Kubernetes Cluster
## Overview
This repository documents the development and deployment of a machine learning (ML) web application for message classification (spam/ham). The project demonstrates an end-to-end ML workflow, from model training to serving the model via a containerized web application on a Kubernetes cluster. The system is designed to handle high loads with up to 50,000 users and 5,000 simultaneous requests.
The project integrates security, scalability, and resilience considerations, including vulnerability assessment and mitigation, stress testing, and protective measures against common web attacks.

## Objectives
The main objectives of this project were:
- Train and save an ML model for message classification (spam/ham)
- Build a web application capable of performing real-time predictions
- Containerize the application using Docker
- Assess and mitigate security vulnerabilities in the Docker environment
- Deploy the web application on a Kubernetes cluster with optimized resource allocation
- Perform stress tests to evaluate scalability and reliability
- Implement security measures to protect the application against cyber-attacks, including DoS threats

## Scope
The project covers:
- ML model development and evaluation
- Web application design for user interaction
- Containerization and local Docker deployment
- Kubernetes cluster creation and management (control and worker nodes)
- Performance and stress testing of the deployed application
- Security analysis and mitigation strategies
